Copycat Website Project

I did this project as I felt like it was a good test of my skills to see whether or not I could create a copy of a website. As for what website I chose, I decided to recreate the Deltamth Sign-in page which was categorized as not being too easy and not being too hard. Based on the final design, it didn't necessarily came out like it was supposed to since I ran out of time due to work from other classes but I still say that it definitely looks similar to the orginal website of Deltamath. Over the course of my time in class, I was able to create the very basics elements of the website including the image and the boxes to hold the text like on the website. On the last day I worked on this project, I learned a useful skill which is being able to create textfields when creating survey type websites as well as replicating what was required to sign in to Deltamath. Overall, if I hadn't run out of time, this could have been way better than it is now but for what it is, it is not bad.
